# Keybinds
|    Keybind      |             Command                  |
|:---------------:|:------------------------------------:|
|   super + Enter |              Term                    |
|    super + r    |             Launcher                 |
|    super + c    |            Clipboard                 |
|    super + m    |              Music                   |
| super + {a,s,d} | Floating, Tiled or Fullscreen state  |
| super + q       |          Close Window                |
|super + Shift + r|            Restart X                 |
|     super + f   |           Change focus               |

*super = Windows Key
